# 👕 AI Virtual Try-On Studio

An AI-powered web application that allows users to upload their image and virtually try on clothing items in real-time.

This project combines computer vision techniques (pose estimation, object detection) with generative AI to simulate realistic clothing overlays.

---

## 🚀 Features

- Upload user image
- Select clothing from catalog
- AI-based virtual try-on
- Before/After comparison
- Download final output
- Fast and responsive UI

---

## 🧠 Tech Stack

- Frontend: React / Streamlit
- Backend: API-based processing
- AI Models:
  - Pose Estimation (MediaPipe)
  - Object Detection (YOLO)
  - AI Try-On (Gemini / Replicate)

---

## 🎯 Use Case

This project demonstrates how AI can be applied in fashion e-commerce for virtual fitting rooms, reducing return rates and improving user experience.
